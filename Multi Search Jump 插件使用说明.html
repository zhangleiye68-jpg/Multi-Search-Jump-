Multi Search Jump 插件使用说明

Multi Search Jump 是一个浏览器搜索插件。你输入一次关键词，它会按照你的设置，同时打开多个网站的搜索结果页。

1. 安装插件

你拿到的安装包通常是一个压缩包，比如 Multi Search Jump.zip。

安装步骤：

  1. 先把安装包解压。
  2. 解压后会得到一个插件文件夹。
  3. 打开这个文件夹，确认里面能看到 manifest.json。
  4. 打开 Chrome 或 Edge。
  5. 进入扩展管理页：
    • Chrome：chrome://extensions
    • Edge：edge://extensions
  6. 开启「开发者模式」。
  7. 点击「加载已解压的扩展程序」。
  8. 选择刚才解压出来的插件文件夹。
  9. 浏览器右上角出现 Multi Search Jump 图标后，安装完成。

不要直接选择压缩包本身。浏览器需要加载的是「解压后的文件夹」。

安装完成后，不要删除或移动这个插件文件夹。浏览器会从这个文件夹读取插件文件，移动后可能导致插件失效。

如果你是在开发者电脑上直接安装，可以选择项目文件夹：

  1. 打开 Chrome 或 Edge。
  2. 进入扩展管理页：
    • Chrome：chrome://extensions
    • Edge：edge://extensions
  3. 开启「开发者模式」。
  4. 点击「加载已解压的扩展程序」。
  5. 选择这个项目文件夹：/Users/kc/Documents/vscode-test
  6. 浏览器右上角出现 Multi Search Jump 图标后，安装完成。

2. 最常用的搜索方式

  1. 点击浏览器右上角的 Multi Search Jump 图标。
  2. 在输入框里输入关键词。
  3. 按 Enter，或点击「搜索」。
  4. 插件会自动打开已启用网站的搜索结果页。

默认支持这些网站：

  • Google
  • X
  • Facebook
  • TikTok
  • 小红书
  • 抖音
  • 微博
  • 知乎
  • 哔哩哔哩

3. 固定搜索面板

如果不想每次都重新点插件弹窗：

  1. 点击插件图标。
  2. 点击「固定」。
  3. 浏览器右侧会打开固定搜索面板。
  4. 以后可以直接在右侧面板里输入关键词搜索。

4. 设置搜索网站

  1. 点击插件图标。
  2. 点击「设置」。
  3. 在「搜索网站与顺序」里调整网站。

使用规则：

  • 开关打开：搜索时会打开这个网站。
  • 开关关闭：搜索时不会打开这个网站。
  • 拖动网站条目：调整搜索结果页打开顺序。
  • 已开启的网站会排在未开启的网站前面。

5. Google 搜索模式

在设置页里可以切换 Google 的搜索类型：

  • 开启「Google 图片搜索」：Google 打开图片搜索结果。
  • 关闭「Google 图片搜索」：Google 打开普通网页搜索结果。

6. 是否关闭上次搜索结果

设置页里有「新搜索前关闭上次结果」开关。

  • 开启：搜索新关键词时，会自动关闭上一次插件打开的搜索标签组。
  • 关闭：搜索新关键词时，上一次的搜索标签页会保留。

7. 搜索历史记录

搜索栏下面会显示最近 5 条搜索记录。

使用方式：

  • 点击历史关键词：重新搜索这个关键词。
  • 点击关键词旁边的 ×：删除这一条历史记录。

设置页底部会显示全部历史记录。历史记录不做去重，你搜过什么，就按实际搜索记录保存什么。

8. 选中文字后搜索

你也可以不用手动输入关键词。

方法一：右键搜索

  1. 在网页里选中一段文字。
  2. 右键点击选中的文字。
  3. 点击「用 Multi Search Jump 搜索」。
  4. 插件会自动打开搜索结果页。

方法二：快捷键搜索

  1. 在网页里选中一段文字。
  2. 按快捷键 Alt+Shift+S。
  3. 插件会自动搜索选中的文字。

如果想修改快捷键：

  1. 打开插件设置页。
  2. 点击「设置快捷键」。
  3. 在浏览器快捷键页面修改。

9. 注意事项

  • X、Facebook、TikTok 等平台的结果，取决于你当前浏览器是否已登录对应账号。
  • 插件只负责打开搜索结果页，不抓取内容，不绕过平台限制。
  • 浏览器内部页面，比如 chrome://extensions，通常不能使用选中文字搜索。
  • 修改插件代码后，需要在扩展管理页点击「刷新」让新版本生效。
