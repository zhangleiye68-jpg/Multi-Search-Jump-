# Multi Search Jump 插件使用说明

Multi Search Jump 是一个浏览器搜索插件。你输入一次关键词，它会按照你的设置，同时打开多个网站的搜索结果页。

安装完成后，也可以在插件设置页的「使用说明」模块查看常用操作。

## 1. 安装插件

你拿到的安装包通常是一个压缩包，比如 `Multi Search Jump.zip`。

安装步骤：

1. 先把安装包解压。
2. 解压后会得到一个插件文件夹。
3. 打开这个文件夹，确认里面能看到 `manifest.json`。
4. 打开 Chrome 或 Edge。
5. 进入扩展管理页：
   - Chrome：`chrome://extensions`
   - Edge：`edge://extensions`
6. 开启「开发者模式」。
7. 点击「加载已解压的扩展程序」。
8. 选择刚才解压出来的插件文件夹。
9. 浏览器右上角出现 Multi Search Jump 图标后，安装完成。

不要直接选择压缩包本身。浏览器需要加载的是「解压后的文件夹」。

安装完成后，不要删除或移动这个插件文件夹。浏览器会从这个文件夹读取插件文件，移动后可能导致插件失效。

如果你是在开发者电脑上直接安装，可以选择本仓库里的 `extension/` 文件夹：

1. 打开 Chrome 或 Edge。
2. 进入扩展管理页：
   - Chrome：`chrome://extensions`
   - Edge：`edge://extensions`
3. 开启「开发者模式」。
4. 点击「加载已解压的扩展程序」。
5. 选择这个仓库里的 `extension/` 文件夹。
6. 浏览器右上角出现 Multi Search Jump 图标后，安装完成。

## 2. 最常用的搜索方式

1. 点击浏览器右上角的 Multi Search Jump 图标。
2. 在输入框里输入关键词。
3. 按 Enter，或点击「搜索」。
4. 插件会自动打开已启用网站的搜索结果页。

新安装后，默认启用 TikTok、Google 图片、X 和 Facebook，并按这个顺序打开；Google 默认使用过去 24 小时筛选。

设置页里仍然可以开启这些网站：

- Google
- X
- Facebook
- TikTok
- Instagram
- Reddit
- 小红书
- 抖音
- 微博
- 知乎
- 哔哩哔哩

## 3. 侧边栏显示

如果不想每次都重新点插件弹窗：

1. 点击插件图标。
2. 点击右上角的侧边栏图标。
3. 以后可以直接在右侧面板里输入关键词搜索。

## 4. 设置搜索网站

1. 点击插件图标。
2. 点击搜索框右上角的设置按钮。
3. 在「搜索」里的「搜索网站与顺序」调整网站。

使用规则：

- 开关打开：搜索时会打开这个网站。
- 开关关闭：搜索时不会打开这个网站。
- 拖动网站条目：调整搜索结果页打开顺序。
- 已开启的网站会排在未开启的网站前面。

## 5. Google 搜索模式

在设置页里可以切换 Google 的搜索类型：

- 默认：Google 图片搜索，并使用过去 24 小时筛选。
- 开启「Google 图片搜索」：Google 打开图片搜索结果。
- 关闭「Google 图片搜索」：Google 打开普通网页搜索结果。
- 开启「使用过去 24 小时搜索」：无论 Google 是普通网页搜索还是图片搜索，都会显示过去 24 小时结果。

## 6. 是否关闭上次搜索结果

设置页里有「新搜索前关闭上次结果」开关。

- 开启：搜索新关键词时，会自动关闭上一次插件打开的搜索标签组。
- 关闭：搜索新关键词时，上一次的搜索标签页会保留。

## 7. 搜索历史记录

开启「小窗口显示历史」后，搜索栏下面会显示最近 5 条搜索记录。

使用方式：

- 点击历史关键词：重新搜索这个关键词。
- 点击关键词旁边的 `×`：删除这一条历史记录。

小窗口只显示最近 5 条，不通过上滑下滑展开更多。「搜索」里的「历史记录」会显示全部历史记录，可以查找、重新搜索、删除单条记录或清空全部历史。历史记录不做去重，你搜过什么，就按实际搜索记录保存什么。

## 8. 选中文字后搜索

你也可以不用手动输入关键词。

方法一：右键搜索

1. 在网页里选中一段文字。
2. 右键点击选中的文字。
3. 点击「用 Multi Search Jump 搜索」。
4. 插件会自动打开搜索结果页。

方法二：快捷键搜索

1. 在网页里选中一段文字。
2. 按快捷键 `Alt+1`；Mac 上默认是 `Option+1`。
3. 插件会自动搜索选中的文字。

如果想修改快捷键：

1. 打开插件设置页。
2. 点击「设置快捷键」。
3. 在浏览器快捷键页面修改。

## 9. 支持下载的网站

支持下载的网站列表统一放在设置页，不再单独打开一个工具页。

查看方式：

1. 点击浏览器右上角的 Multi Search Jump 图标。
2. 点击右上角的设置按钮。
3. 在「下载」模块里找到「支持下载的网站」。
4. 「下载悬浮图标」开关控制支持网站页面上的本地下载入口。

它可以在本机处理当前浏览器页面可获得的数据，比如媒体、评论、表格和字幕下载。下载文件会进入浏览器下载目录，文件名会尽量保留平台、标题、类型、分辨率和时间等信息。

边界说明：

- 不需要登录 DataTool。
- 不提供购买、会员或云端上传能力。
- 不恢复云端语音转文字。
- 不会把下载记录、搜索历史或页面数据保存到 GitHub 仓库文件里。
- 能否拿到数据，取决于当前网页和平台接口本身是否提供这些数据，以及你当前浏览器是否已登录对应平台。

## 10. 注意事项

- X、Facebook、TikTok、Instagram、Reddit 等平台的结果，取决于你当前浏览器是否已登录对应账号。
- 插件只负责打开搜索结果页，不抓取内容，不绕过平台限制。
- 本地工具会新增下载、cookies、较大本地存储、网络拦截规则和多个平台访问权限，用来在本机下载页面可获得的数据并阻断 DataTool 云端/付费/上传接口。
- 浏览器内部页面，比如 `chrome://extensions`，通常不能使用选中文字搜索。
- 修改插件代码后，需要在扩展管理页点击「刷新」让新版本生效。
