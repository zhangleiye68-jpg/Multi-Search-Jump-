# Multi Search Jump 插件使用说明

Multi Search Jump 是一个浏览器插件，主要用于多平台搜索、TikTok 字幕/数据辅助，以及支持网站的本地下载入口。

## 1. 支持几个窗口？

插件主要有 4 个窗口或界面。

| 窗口 | 在哪里打开 | 主要用途 |
| --- | --- | --- |
| 插件小窗口 | 点击浏览器右上角插件图标 | 输入关键词、发起搜索、打开侧边栏、打开设置、查看最近历史 |
| 侧边栏 | 点击小窗口里的侧边栏按钮 | 固定在浏览器右侧，用来连续搜索；在 TikTok 页面也可显示字幕看板 |
| 设置页 | 点击小窗口或侧边栏里的设置按钮 | 管理搜索、字幕、下载和使用说明 |
| TikTok 字幕悬浮窗 | 在 TikTok 页面点击 `CC` 按钮 | 查看字幕、翻译、视频介绍和视频数据 |

另外，支持下载的网站页面会出现「下载悬浮图标」。它不是一个窗口，而是页面上的下载入口。

## 2. 插件小窗口有哪些按钮？

| 按钮 | 功能 |
| --- | --- |
| 搜索 | 根据输入的关键词，打开已启用网站的搜索结果页 |
| 侧边栏按钮 | 打开浏览器右侧的固定搜索面板 |
| 设置按钮 | 打开插件设置页 |
| 历史关键词 | 重新搜索这条历史关键词 |
| `×` | 删除对应的一条搜索历史 |

新安装后，默认启用 TikTok、Google 图片、X 和 Facebook，并按这个顺序打开；Google 默认使用过去 24 小时筛选。

支持搜索的网站包括：Google、X、Facebook、TikTok、Instagram、Reddit、小红书、抖音、微博、知乎、哔哩哔哩。

## 3. 侧边栏有哪些按钮？

| 按钮 | 功能 |
| --- | --- |
| 搜索 | 在右侧面板里直接发起多平台搜索 |
| 设置按钮 | 打开插件设置页 |
| 原版 | TikTok 字幕看板只显示原始字幕 |
| 原+中 | TikTok 字幕看板同时显示原文和中文翻译 |
| 中文 | TikTok 字幕看板只显示中文翻译 |
| 打开字幕悬浮窗 | 把 TikTok 字幕看板切回页面上的悬浮窗 |
| `-` / `+` | 缩小或放大字幕文字 |
| 刷新字幕 | 重新读取当前 TikTok 视频字幕 |
| 复制全部 | 复制当前字幕内容 |
| 复制原文 | 复制视频介绍原文 |

侧边栏适合连续搜索，也适合在 TikTok 页面右侧查看字幕，不让字幕面板盖住页面。

## 4. 设置页有哪些按钮和开关？

设置页分为 4 个模块：搜索、字幕、下载、使用说明。

### 4.1 搜索模块

| 按钮或开关 | 功能 |
| --- | --- |
| 直接搜索里的搜索按钮 | 在设置页直接发起搜索 |
| 新搜索前关闭上次结果 | 新搜索时自动关闭上一次由插件打开的标签组 |
| 关闭上次结果 | 手动关闭上一次搜索结果标签组 |
| 将中文翻译成英文后搜索 | 输入中文时先转成英文再搜索 |
| Google 图片搜索 | 开启后 Google 打开图片搜索；关闭后打开普通网页搜索 |
| 使用过去 24 小时搜索 | Google 只显示过去 24 小时结果 |
| 搜索网站开关 | 控制某个网站是否参与搜索 |
| 网站条目拖动排序 | 调整搜索结果页的打开顺序 |
| 小窗口显示历史 | 控制插件小窗口是否显示最近 5 条历史 |
| 侧边栏显示 | 打开或关闭浏览器右侧搜索面板 |
| 设置快捷键 | 打开浏览器快捷键页面，修改选中文字搜索快捷键 |
| 清空全部历史 | 删除所有搜索历史 |
| 历史记录里的重新搜索 | 用某条历史关键词再次搜索 |
| 历史记录里的删除 | 删除某一条历史 |

选中文字搜索也支持两个入口：

- 右键菜单：选中文字后右键点击「用 Multi Search Jump 搜索」。
- 快捷键：Windows 默认 `Alt+1`，Mac 默认 `Option+1`。

### 4.2 字幕模块

| 开关 | 功能 |
| --- | --- |
| 自动打开字幕板 | 进入 TikTok 并检测到完整字幕源时，自动打开字幕板 |
| 视频列表显示数据指标 | 在支持的 TikTok 视频列表卡片上显示数据指标 |
| 非英语警示 | 非英语视频显示「非英内容」提醒 |

视频列表数据指标包括：权重评价、每小时点赞量、总播放量、每小时播放量和发布时长。

### 4.3 下载模块

| 按钮或开关 | 功能 |
| --- | --- |
| 下载悬浮图标 | 控制支持网站页面上的本地下载入口是否显示 |
| 支持下载的网站 | 查看当前支持下载入口的网站列表 |

支持下载的网站包括：TikTok、Douyin、Instagram、X / Twitter、Facebook、Bilibili、小红书、Kwai、快手、新片场、Vimeo。

## 5. TikTok 页面有哪些按钮？

| 按钮或入口 | 功能 |
| --- | --- |
| `CC` | 打开 TikTok 字幕悬浮窗 |
| 原版 / 原+中 / 中文 | 切换字幕显示方式 |
| `-` / `+` | 调整字幕文字大小 |
| 刷新字幕 | 重新读取当前视频字幕 |
| 复制全部 | 复制字幕内容 |
| 复制原文 | 复制视频介绍原文 |
| 筛选 | 打开视频列表筛选面板 |
| 收起 | 收起筛选面板 |
| 全部 / 高 / 中 / 低 | 按爆款评价筛选视频 |
| 近24小时 / 近2天 / 近7天 / 不限时间 | 按发布时间范围筛选视频 |
| 加载更多 | 尝试加载更多视频卡片 |
| 重置 | 清空筛选条件 |
| 应用筛选 | 按当前条件筛选并排序视频列表 |

TikTok 字幕功能只读取页面已有字幕或页面接口返回的字幕，不做云端语音转文字。

## 6. 下载页面有哪些按钮？

在支持网站上，主要入口是「下载悬浮图标」。点击后，会根据当前页面能提供的数据显示可下载内容。

可能出现的下载项包括：视频、音频、图片、封面、字幕、评论或页面可导出的数据。

下载文件会进入浏览器默认下载目录。能否下载成功，取决于当前网页、平台接口和你的登录状态。

## 7. 简短注意事项

- 插件只负责打开搜索结果页，不绕过平台限制。
- X、Facebook、TikTok、Instagram、Reddit 等平台结果，可能取决于当前浏览器是否已登录。
- 搜索历史保存在浏览器本地，历史不去重，最多保留 1000 条。
- 下载功能是本地入口，不需要登录 DataTool，不提供购买、会员或云端上传能力。
- 浏览器内部页面，比如 `chrome://extensions`，通常不能使用选中文字搜索。
- 修改插件文件后，需要在扩展管理页点击「刷新」让新版本生效。
