# Multi Search Jump 安装与使用教程

版本：1.0.0

这份教程是给安装插件的人看的。Multi Search Jump 是一个 Chrome / Edge 浏览器插件，可以用一个关键词同时打开多个平台的搜索结果，也可以在 TikTok 页面查看字幕、翻译、视频数据和本地下载入口。

## 1. 安装前准备

你需要准备：

- Chrome 或 Edge 浏览器。
- Multi Search Jump 安装包。安装包通常是一个 ZIP 压缩包，解压后里面应当能看到 `extension/` 文件夹。

安装时真正要加载的是 `extension/` 文件夹，因为插件的 `manifest.json` 在这个目录里。

不要选择 ZIP 文件，也不要选择仓库根目录。如果选错，浏览器会提示找不到清单文件，或者插件不能正常加载。

## 2. 解压安装包

1. 找到收到的 Multi Search Jump ZIP 安装包。
2. 右键解压，或双击后选择“解压全部”。
3. 打开解压后的文件夹，确认里面有 `extension/` 文件夹。
4. 打开 `extension/` 文件夹，确认能看到 `manifest.json`。

安装完成后，不要移动或删除解压后的文件夹。浏览器会持续从这个位置读取插件文件。

## 3. 在 Chrome 安装

1. 打开 Chrome。
2. 在地址栏输入 `chrome://extensions` 并回车。
3. 打开右上角“开发者模式”。
4. 点击“加载已解压的扩展程序”。
5. 选择解压后的 `extension/` 文件夹。
6. 点击确认。
7. 页面中出现 Multi Search Jump，说明安装完成。

如果浏览器提示权限确认，选择允许即可。插件需要这些权限来打开搜索结果页、使用侧边栏、保存本地设置、读取 TikTok 字幕信息，以及在支持的网站显示下载入口。

## 4. 在 Edge 安装

1. 打开 Edge。
2. 在地址栏输入 `edge://extensions` 并回车。
3. 打开左侧或右侧的“开发人员模式”。
4. 点击“加载解压缩的扩展”或“加载已解压的扩展程序”。
5. 选择解压后的 `extension/` 文件夹。
6. 点击确认。
7. 页面中出现 Multi Search Jump，说明安装完成。

Edge 的按钮文案可能因版本不同略有差异，但核心动作都是加载已解压的 `extension/` 文件夹。

## 5. 固定插件图标

安装后建议把插件固定到浏览器右上角：

1. 点击浏览器右上角的扩展图标。
2. 找到 Multi Search Jump。
3. 点击固定按钮。

固定后，浏览器右上角会一直显示 Multi Search Jump 图标，后续搜索会更方便。

## 6. 基础搜索用法

1. 点击浏览器右上角的 Multi Search Jump 图标。
2. 在输入框输入关键词。
3. 按 Enter，或点击“搜索”。
4. 插件会按照已启用的网站顺序打开搜索结果页。

新安装后默认启用 TikTok、Google 图片、X 和 Facebook。你可以在设置页开启或关闭更多网站，也可以拖动调整打开顺序。

支持的搜索网站包括 Google、X、Facebook、TikTok、Instagram、Reddit、小红书、抖音、微博、知乎和哔哩哔哩。

## 7. 使用侧边栏

侧边栏适合连续搜索，也适合在 TikTok 页面右侧查看字幕看板。

打开方式：

1. 点击 Multi Search Jump 图标。
2. 点击小窗口右上角的侧边栏按钮。
3. 浏览器右侧会打开固定面板。

侧边栏里可以直接输入关键词搜索，也可以打开设置页。如果当前页面是 TikTok 视频页，侧边栏还可以显示 TikTok 字幕看板。

## 8. 设置搜索网站和历史

1. 点击 Multi Search Jump 图标。
2. 点击设置按钮。
3. 在“搜索”模块中调整搜索网站。

常用设置包括：

- 开启或关闭某个搜索网站。
- 拖动网站条目，调整搜索结果页打开顺序。
- 开启或关闭 Google 图片搜索。
- 开启或关闭 Google 过去 24 小时筛选。
- 开启或关闭中文关键词自动翻译成英文后搜索。
- 控制小窗口是否显示最近搜索历史。
- 清空或删除搜索历史。

搜索历史保存在浏览器本地，不会上传到云端。历史记录不去重，最多保留 1000 条。

## 9. 选中文字搜索

插件支持在网页上选中文字后快速搜索。

右键搜索：

1. 在网页中选中一段文字。
2. 右键点击选中文字。
3. 点击“用 Multi Search Jump 搜索”。

快捷键搜索：

- Windows 默认快捷键是 `Alt+1`。
- Mac 默认快捷键是 `Option+1`。

如果想修改快捷键，可以在设置页点击“设置快捷键”，进入浏览器快捷键管理页面调整。

## 10. TikTok 字幕看板

打开 TikTok 视频页后，页面上会出现 `CC` 按钮。

常用操作：

1. 点击 `CC`，打开 TikTok 字幕悬浮窗。
2. 点击“原版”，只看原始字幕。
3. 点击“原+中”，同时看原文和中文翻译。
4. 点击“中文”，只看中文翻译。
5. 点击 `-` 或 `+`，调整字幕文字大小。
6. 点击“刷新字幕”，重新读取当前视频字幕。
7. 点击“复制全部”，复制当前字幕内容。
8. 点击“复制原文”，复制视频介绍原文。

如果想把字幕看板放到浏览器侧边栏里，可以点击字幕悬浮窗上的侧边栏按钮，或先打开 Multi Search Jump 侧边栏。

TikTok 字幕功能只读取页面已有字幕或页面接口返回的字幕，不做云端语音转文字。没有字幕源的视频，可能无法显示字幕。

## 11. TikTok 数据和筛选

在支持的 TikTok 视频列表页面，插件会显示视频数据指标，例如：

- 爆款评价。
- 每小时点赞量。
- 总播放量。
- 每小时播放量。
- 发布时长。

如果页面上出现“筛选”按钮，可以点击打开筛选面板。常用操作包括：

- 按高、中、低爆款评价筛选。
- 按近 24 小时、近 2 天、近 7 天或不限时间筛选。
- 输入最低点赞量、播放量等条件。
- 点击“加载更多”尝试加载更多视频卡片。
- 点击“应用筛选”排序和筛选视频。
- 点击“重置”清空筛选条件。

筛选只整理当前页面已经加载或继续加载出来的视频卡片，不会绕过 TikTok 平台限制。

## 12. 下载悬浮图标

在支持的网站页面上，可能会出现下载悬浮图标。点击后，插件会根据当前页面可获得的数据展示可下载内容。

可能出现的下载内容包括：

- 视频。
- 音频。
- 图片。
- 封面。
- 字幕。
- 评论。
- 页面可导出的数据。

支持下载入口的网站包括 TikTok、Douyin、Instagram、X / Twitter、Facebook、Bilibili、小红书、Kwai、快手、新片场和 Vimeo。

下载文件会进入浏览器默认下载目录。能否下载成功，取决于当前网页、平台接口和你的登录状态。

## 13. 更新插件

如果你收到新版本安装包：

1. 解压新版本 ZIP。
2. 打开浏览器扩展管理页：
   - Chrome：`chrome://extensions`
   - Edge：`edge://extensions`
3. 找到 Multi Search Jump。
4. 如果你想保留原来的插件条目，可以把原来的插件文件夹替换成新版本文件夹，然后点击“刷新”。
5. 如果你重新选择了新文件夹，可以先移除旧插件，再点击“加载已解压的扩展程序”，选择新版本的 `extension/` 文件夹。

更新后如果页面里旧功能没有变化，回到扩展管理页点击“刷新”，然后刷新正在使用的网站页面。

## 14. 常见问题

### 浏览器提示找不到 manifest.json

通常是选错了文件夹。请重新选择解压后的 `extension/` 文件夹，不要选择 ZIP 文件，也不要选择仓库根目录。

### 插件图标不见了

点击浏览器右上角扩展图标，找到 Multi Search Jump，然后重新固定。

### 搜索结果没有全部打开

检查设置页里对应网站是否开启。某些平台也可能因为未登录、地区限制或网页策略而显示不同结果。

### TikTok 没有字幕

并不是所有视频都有可读取字幕。可以点击“刷新字幕”重试。如果视频本身没有字幕源，插件不会做语音转文字。

### 下载按钮没有出现

先确认当前网站在支持列表中，再检查设置页“下载悬浮图标”是否开启。部分页面需要登录平台账号后才能读取可下载数据。

### 修改文件后没有生效

打开 `chrome://extensions` 或 `edge://extensions`，找到 Multi Search Jump，点击“刷新”，再刷新网页。

## 15. 卸载插件

1. 打开 `chrome://extensions` 或 `edge://extensions`。
2. 找到 Multi Search Jump。
3. 点击“移除”。

卸载后，插件功能会停止。你也可以删除本地解压出来的安装包文件夹。
