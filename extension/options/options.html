    使用说明

    查看插件支持的窗口、主要按钮和每个入口的大致功能。

    支持的窗口

    • 插件小窗口：输入关键词、发起搜索、打开侧边栏、打开设置、查看最近历史。
    • 侧边栏：固定在浏览器右侧，用来连续搜索；在 TikTok 页面也可显示字幕看板。
    • 设置页：管理搜索、字幕、下载和使用说明。
    • TikTok 字幕悬浮窗：在 TikTok 页面点击 CC 后查看字幕、翻译、视频介绍和视频数据。

    插件小窗口按钮

    • 「搜索」：根据输入关键词打开已启用网站的搜索结果页。
    • 侧边栏按钮：打开浏览器右侧的固定搜索面板。
    • 设置按钮：打开插件设置页。
    • 历史关键词：重新搜索这条历史关键词；旁边的 × 可以删除这一条历史。

    新安装后，默认启用 TikTok、Google 图片、X 和 Facebook，并按这个顺序打开；Google 默认使用过去 24 小时筛选。

    侧边栏按钮

    • 「搜索」：在右侧面板里直接发起多平台搜索。
    • 设置按钮：打开插件设置页。
    • 「原版 / 原+中 / 中文」：切换 TikTok 字幕显示方式。
    • 「打开字幕悬浮窗」：把 TikTok 字幕看板切回页面上的悬浮窗。
    • 「- / +」：缩小或放大字幕文字。
    • 「刷新字幕 / 复制全部 / 复制原文」：重新读取字幕、复制字幕或复制视频介绍原文。

    设置页按钮和开关

    • 「新搜索前关闭上次结果」：新搜索时自动关闭上一次由插件打开的标签组。
    • 「关闭上次结果」：手动关闭上一次由插件打开的搜索结果标签组。
    • 「将中文翻译成英文后搜索」：输入中文时先转成英文再搜索。
    • 「Google 图片搜索 / 使用过去 24 小时搜索」：控制 Google 搜索类型。
    • 「搜索网站与顺序」：开启或关闭平台，并拖动调整打开顺序。
    • 「小窗口显示历史」：小窗口只显示最近 5 条；「搜索」里的「历史记录」会显示全部历史记录。
    • 「设置快捷键」：修改选中文字搜索快捷键，默认是 Alt+1，Mac 默认是 Option+1。
    • 「自动打开字幕板 / 视频列表显示数据指标 / 非英语警示」：控制 TikTok 字幕和数据指标。
    • 「下载悬浮图标」：控制支持网站页面上的本地下载入口。

    TikTok 页面按钮

    • 「CC」：打开 TikTok 字幕悬浮窗。
    • 「原版 / 原+中 / 中文」：切换字幕显示方式。
    • 「刷新字幕 / 复制全部 / 复制原文」：重新读取字幕、复制字幕或复制视频介绍原文。
    • 「筛选」：打开视频列表筛选面板。
    • 「全部 / 高 / 中 / 低」：按爆款评价筛选视频。
    • 「近24小时 / 近2天 / 近7天 / 不限时间」：按发布时间范围筛选视频。
    • 「加载更多 / 重置 / 应用筛选」：加载更多卡片、清空筛选条件或应用当前筛选。

    下载页面入口

    支持网站页面上的「下载悬浮图标」会显示本地下载入口，可能包含视频、音频、图片、封面、字幕、评论或页面可导出的数据。能否下载成功，取决于当前网页、平台接口和登录状态。

    注意事项

    • X、Facebook、TikTok、Instagram、Reddit 等平台结果，取决于当前浏览器是否已登录对应账号。
    • 插件只负责打开搜索结果页，不绕过平台限制。
    • 下载功能是本地入口，不需要额外登录旧工具服务，不提供购买、会员或云端上传能力。
    • 浏览器内部页面，比如 chrome://extensions，通常不能使用选中文字搜索。
    • 修改插件代码后，需要在扩展管理页点击「刷新」让新版本生效。
